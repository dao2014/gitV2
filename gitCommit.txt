-----版本回退

git reset --hard HEAD~    可以使用数字 代替  ~1 回退的版本
git reset --hard ID       可以指定版本 包括前面的版本. 只要记住 ID

git log --查看日志

git reflog --查看命令提交日志  可以通过这 回复到前一个版本
